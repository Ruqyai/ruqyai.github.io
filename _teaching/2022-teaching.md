#### Commands of docker compose

* build the docker image
```bash
docker compose build  
```  
* run all 

```bash
docker compose up  
```  
* stop and remove all 

```bash
docker compose down 
```  
* clean 
```bash
docker volume rm $(docker volume ls -q)
docker image rm $(docker image ls -q)
docker system prune
```
